# 116 會考總複習筆記 + 測驗系統

為國九學生準備**民國 116 年（2027/5）國中教育會考**的純前端複習網站：跨版本總複習筆記、語音朗讀、選擇題測驗、月／週／日進度與考前 60 天衝刺，全部離線可用。

## 怎麼開啟
直接**雙擊 `index.html`**（建議用 Edge 或 Chrome，語音較自然）。不需安裝、不需網路、不需伺服器。

## 功能
- **儀表板** `index.html`：距會考天數、目前階段、連續達成天數、測驗平均、**作答/答對/答錯統計**、今日任務、各科進度、打卡日曆、進度匯出/匯入。重設進度**需密碼**（`config.js` 的 `resetPassword`，預設 `0000`）。
- **進度計畫** `schedule.html`：六階段時間軸、月／週重點、**每日計畫**（可選日期查看）；考前 60 天自動切換衝刺樣式。
- **各科筆記** `subject.html`：跨版本主題筆記，附**橘色粗心提醒**與**圖解（內嵌 SVG）**；🔊 逐段或整頁朗讀、語速可調。
- **測驗** `quiz.html`：選擇題即時對錯＋詳解；標 ⚠ 易錯題送出前**再讀一次題目**；結算顯示答對/答錯題數。
  - 份量：**每日各科 10 題**（`?set=daily`）、**週六週測 20 題**（`?set=weekly`）、**每月 15／28 號各科綜合 30 題**（`?set=monthly`）；另有單一主題與整科綜合（`?topic=all`）。抽題依日期可重現。
- **錯題本** `mistakes.html`：自動收集答錯題（含上述各種測驗），附詳解與「為何容易看錯」。

## 內容狀態
六科皆已建置筆記＋題庫＋圖解（共約 275 題）：
- 數學 20、國文 16、英語 18 單元；社會（歷史/地理/公民）31、自然（理化/生物/地科）31 單元；寫作測驗 1。
- **每日核心輪播**：每個讀書日自動指派 數/國/英 各一主題筆記＋對應份量測驗（週日休息）。

## 擴充內容
編輯 `js/data/*.js`（就是 JSON 物件）。格式與範例見 **`js/data/schema.md`**，最完整的參考是 **`js/data/math.js`**。

## 設定
- 改考試日期：`js/data/config.js` 的 `examDate`（會考正式公告後改這一個值）。
- 想用更擬真的語音：在筆記段落加 `audioSrc: "audio/xxx.mp3"`，即會改播音檔取代瀏覽器內建語音。

## 進度資料
存在瀏覽器 `localStorage`（單機、離線）。按儀表板「重設進度」可清空。

## 自我檢查
任一頁面網址加 `?selftest=1`（例：`index.html?selftest=1`），開瀏覽器 Console 看日期計算／計分／streak 的斷言結果。
