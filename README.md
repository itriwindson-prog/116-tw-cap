# 116 會考總複習筆記 + 測驗系統（StudySync）

為國九學生準備**民國 116 年（2027/5）國中教育會考**的純前端複習網站：跨版本總複習筆記、語音朗讀、選擇題測驗、月／週／日進度與考前 60 天衝刺。**純前端、可離線、可安裝成 App**。

---

## 怎麼使用

**① 電腦（最快）**
直接**雙擊 `index.html`**（建議 Edge / Chrome，語音較自然）。不需安裝、網路、伺服器。

**② 平板 / 手機（給學生，推薦）**
把網站掛成一個網址（見下方〈部署〉），學生用瀏覽器開、**加到主畫面**後就像一支 App，全螢幕、可離線。

---

## 部署到網路（給學生用）— GitHub Pages

> 一次設定，之後更新只要 `git push`。

1. 到 github.com 建一個**空 repo**（例：`cap-studysync`），**Public**，不要勾 README / .gitignore。
2. 在專案資料夾的終端機執行（網址換成你的）：
   ```bash
   git remote add origin https://github.com/<你的帳號>/cap-studysync.git
   git push -u origin master
   ```
   跳出登入時用瀏覽器授權即可。
3. **開啟 Pages**：repo → **Settings → Pages** → Source 選「Deploy from a branch」→ branch `master`、資料夾 `/ (root)` → **Save**。等約 1 分鐘。
4. 你的網址：`https://<你的帳號>.github.io/cap-studysync/`

**日後更新內容**：改完 →
```bash
git add -A && git commit -m "更新內容" && git push
```
學生**下一次有網路開啟 App 時自動更新**（service worker 採 stale-while-revalidate，不必手動改版本號；只有改動 `sw.js` 的快取清單時才需把 `CACHE` 版本 +1）。

## 學生安裝到平板

**iPad / iPhone（務必用 Safari）**
1. Safari 開老師給的網址。
2. 點下方「分享」鈕 → **加入主畫面** → 命名「會考總複習」→ 加入。
3. 桌面出現「會考」圖示，點開即用（全螢幕、可離線）。
   - iOS 的 Chrome 不能加主畫面，請用 Safari。

**Android（Chrome）**
1. Chrome 開網址 → 通常會跳「安裝應用程式」；或右上 ⋮ → **安裝應用程式／加到主畫面**。
2. 桌面出現圖示，可離線開。

> 第一次需連網一次（下載快取），之後**沒有網路也能開**。

## 給學生的提醒
- **進度存在各自的平板**（測驗分數、打卡、錯題本），不互通、不上傳雲端。
- 換平板或清除瀏覽器資料會清空 → 用儀表板右上**「匯出」**存一個 JSON 備份，新平板用**「匯入」**還原。

---

## 功能

- **儀表板** `index.html`（主入口）
  - 第一列三卡：距會考天數／目前階段、連續達成天數、測驗平均＋作答/答對/答錯統計。
  - **各科進度**、**當月進度月曆**（主要入口）、折疊式**本月計畫**與**本週計畫**、**今日任務**。
  - 月曆：**綠底=當天任務完成、紅底=過期未完成、今天閃爍**（尊重「減少動態效果」設定）；**點任一天進入當天任務**；可上／下月切換與回本月。
  - **打卡與連續天數**靠今日任務卡的「今天的任務都完成了」勾選驅動。
  - 進度匯出／匯入；重設進度**需密碼**（`js/data/config.js` 的 `resetPassword`，預設 `0000`）。
- **進度計畫** `schedule.html`：**每日計畫**（可選任一天查看，網址帶 `?date=YYYY-MM-DD`）、六階段時間軸、**月計畫**（11 個月逐月折疊，本月／即將開始的月自動展開）。
- **各科筆記** `subject.html`：跨版本主題筆記，附**橘色粗心提醒**與**圖解（內嵌 SVG）**；🔊 逐段或整頁朗讀、語速可調。讀完可直接進入該主題測驗。
- **測驗** `quiz.html`：**交卷制**——點選項只會標記為已選（可隨時改選），**不**即時顯示對錯、**不**跳示警；按**「交卷」**後一次逐題揭曉對錯＋詳解（含易錯題的「為何容易看錯」），未作答題標「未作答」並計為錯，最後結算分數並寫回進度與錯題本。
  - **份量固定、每次隨機抽**：每日各科 **10 題**（`?set=daily`）、週六週測 **20 題**（`?set=weekly`）、每月 15／28 號各科綜合 **30 題**（`?set=monthly`）、單一主題 **10 題**（`?topic=<id>`）、本科綜合 **30 題**。
  - 主題連動：每日以「當天輪播主題」為主、週測限本週 6 個輪播主題、月考為本科綜合。
- **錯題本** `mistakes.html`：自動收集各類測驗答錯的題目，附詳解與「為何容易看錯」，可「重做這個主題」。

## 擴充內容
編輯 `js/data/*.js`（就是 JSON 物件）。格式與範例見 **`js/data/schema.md`**，最完整的參考是 **`js/data/math.js`**。

## 設定
- 改考試日期：`js/data/config.js` 的 `examDate`（會考正式公告後改這一個值）。
- 更擬真語音：在筆記段落加 `audioSrc: "audio/xxx.mp3"`，即改播音檔取代瀏覽器內建語音。

## 進度資料
存在瀏覽器 `localStorage`（前綴 `ss.`，單機、離線）。按儀表板「重設進度」可清空（需密碼）。

## 技術
純 HTML/CSS/原生 JS，無框架、無建置步驟；資料以 JS 內嵌（`window.STUDYSYNC.data.*`）避開 `file://` 的 fetch 限制。離線可安裝靠 `manifest.webmanifest` + `sw.js`（PWA）。

## 自我檢查
任一頁面網址加 `?selftest=1`（例：`index.html?selftest=1`），開瀏覽器 Console 看日期計算／計分／streak 的斷言結果。
